# icons

A comprehensive collection of icons in various formats, designed for applications, dashboards, and general use. This repository organizes icons by their file formats and includes detailed metadata for categorization and tracking.

## Repository Structure

The repository is structured logically to help you find icons and their associated metadata quickly:

- **`dashboard-icons/`**: Contains a curated set of dashboard icons (from Homarr Labs).
- **`metadata/`**: JSON files containing metadata for individual icons, including format info, categories, and update timestamps.
- **Image Formats**:
  - **`ico/`**: Classic Windows Icon format files.
  - **`jpeg/` & `jpg/`**: JPEG image files.
  - **`png/`**: Portable Network Graphics files.
  - **`svg/`**: Scalable Vector Graphics files (ideal for responsive web usage).
  - **`webp/`**: Modern WebP image files for optimized web delivery.

## Usage

You can navigate through the respective format directories (`svg`, `png`, `webp`, etc.) to find the icons you need for your projects. 

If you are building an application that programmatically accesses these icons, you can use the JSON files within the `metadata/` directory to query icon categories, aliases, and updates.

## Contributing

When adding new icons to the repository:
1. Place the image file in the appropriate directory matching its format (e.g., `svg/` for `.svg` files).
2. Create or update the corresponding `.json` file in the `metadata/` directory to maintain consistent categorization.
